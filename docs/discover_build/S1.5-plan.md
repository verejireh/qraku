# S1.5 — Square 미연동 온보딩 카드 (스펙 + 계획)

> 브랜치: `discover_build` · 작성일: 2026-06-08 · 상위: [WORKPLAN.md](./WORKPLAN.md)

## 목표
사장님이 결제설정에서 Square를 골랐지만 아직 연동 안 한 상태일 때, **"단말기 없어도 온라인 사전결제 가능"**과 **가입→활성화→연동 3단계**를 화면에 직접 안내한다. 상세는 기존 가이드([payment_setup_guide.html](../../frontend-react/public/docs/payment_setup_guide.html))로 연결. → S1에서 만든 테이크아웃 버튼이 더 많은 가게에 노출되도록 하는 공급 측 작업.

## 배경
기존 가이드 문서에 이미 "물리端末 불필요"(line 563)·Square 가입·활성화 내용 존재. 문제는 **발견성** — 미연동 화면엔 *"未連携です…"* 한 줄뿐. → 결정 순간에 인라인 카드로 노출만 하면 됨.

## 범위 / File Fence
- 수정: `frontend-react/src/views/AdminPaymentView.jsx` **한 파일만**.
- 불가침: `square_oauth.py`, 결제 로직, 백엔드, 가이드 문서(재작성 안 함).
- i18n 미도입(파일이 전부 JP 하드코딩 — 동일 패턴 유지).
